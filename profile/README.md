# **Check Your Breath**

## **Autores:** Gabriela Dellamora, João Cairuga, José Gouvea, Leonardo Ripes

O projeto é composto 8 repositórios, abaixo está uma breve descrição do conteúdo que compõe cada repositório, para maior esclarecimento sobre cada um, basta acessar seus READMEs que estão dentro de seus domínios.

**Backend:** Repositório que é composto pelos códigos responsáveis pelo funcionamento do backend do projeto, desde o funcionamento do aplicativo em si quanto em relação a API que será utilizada para encapsular os modelos classificadores.

**Frontend:** Repositório onde de localiza todo o código responsável pelo front end do aplicativo desenvolvido.

**CYB.Model.TableClassifier:** Repositório que tem em seu conteúdo toda arquitetura do modelo utilizado para a classificação das anamneses armazenadas em formato tabular

**CYB.Model.ImageClassifier:** Repositório que tem em seu conteúdo toda arquitetura do modelo utilizado para a classificação das imagens

**preprocessed_data:** Repositório composto pelas imagens já segmentadas pelo algortimo SAM

**raw:** Repositório composto pelas imagens brutas obtidas a partir do envio do cliente

**image-preprocessing-notebooks:** Repositório composto pelos notebooks usado para testes de técnicas variadas de pré-processamento das imagens.

**CYB_EDA:** Repositório composto dos notebooks contendo os scripts responsáveis pela análise exploratória dos dados de imagens e tabulares.
